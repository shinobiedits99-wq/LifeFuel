🚀 LifeFuel AI - Smart Health Dashboard
LifeFuel AI is a high-performance, mobile-responsive health tracking application designed with a sleek Glassmorphism UI. It goes beyond simple calorie counting by using logic-driven AI to suggest meals and gamification to keep users engaged.

✨ Key Features
🔐 Secure Authentication: Full user login/signup system powered by Supabase Auth, ensuring private data stays private.

📊 Dynamic Progress Tracking: Real-time calorie logs with a daily reset engine.

📈 Weekly Analytics: Visualized data via Chart.js to track nutritional trends over a 7-day period.

🔥 Gamification System: Built-in "Streak" tracker and "Badge" rewards (Rookie to Legend) to encourage daily logging.

🤖 AI Nutrition Assistant: A smart suggestion engine that analyzes remaining daily calories to recommend the perfect next meal.

🎨 Premium UI/UX: * Glassmorphism cards with backdrop-filter.

3D-animated category selectors (Breakfast, Lunch, Dinner, Snacks).

Neon-glow effects for high visibility.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Keyframe Animations), JavaScript (ES6+).

Database: Supabase (PostgreSQL) with Row Level Security (RLS).

Auth: Supabase Go-True.

Charts: Chart.js.

Hosting: Vercel.
